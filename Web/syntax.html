CSS Syntax Basics

CSS (Cascading Style Sheets) is used to style HTML elements. A CSS rule consists of a selector and a declaration block.

selector {
  property: value;
  property: value;
}

Explanation:

  • Selector: Specifies which HTML element(s) the styles apply to. Example: p, #id, or .class.
  • Property: The CSS property you want to change, like color, font-size, or background-color.
  • Value: The value assigned to the property, e.g., red, 16px, or #fff.

Example:

h1 {
  color: blue;
  font-size: 36px;
  text-align: center;
}

This rule styles all <h1> elements to have blue text, a font size of 36 pixels, and centered alignment.
